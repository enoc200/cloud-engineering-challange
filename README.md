Part 4 — Basic Research Task

1. What is cloud computing?  
Cloud computing is the delivery of computing resources (like servers, storage, databases, networking, and software) over the internet, allowing on-demand access without owning physical hardware.

2. What is DevOps?  
DevOps is a set of practices that combines software development (Dev) and IT operations (Ops) to shorten the development lifecycle, improve collaboration, and deliver high-quality software continuously.

3. Difference between Docker and Kubernetes  
- Docker: A platform to create, package, and run containers for applications.  
- Kubernetes: A system to orchestrate and manage multiple containers across many machines, ensuring scalability, load balancing, and fault tolerance.

4. Why are Linux skills important in cloud engineering?  
Most cloud servers run on Linux because it is stable, secure, flexible, and efficient. Knowing Linux helps in server management, scripting, automation, and troubleshooting in cloud environments.


Part 5 — Problem Solving Question

A cloud server becomes inaccessible. What are 3 things you would check first?  
1. Check if the server is running and not stopped or terminated.  
2. Verify network connectivity (firewalls, security groups, and IP addresses).  
3. Check server logs for errors or crashes to identify the root cause.
